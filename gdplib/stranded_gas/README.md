# Stranded gas production network

This is a multi-period production planning, allocation, and design model involving modular facilities for production of gasoline from stranded gas sources.

Source paper:

> Chen, Q., & Grossmann, I. E. (2019). Economies of Numbers for A Modular Stranded Gas Processing Network: Modeling And Optimization. In *Proceedings of the 9th International Conference on Foundations of Computer-Aided Process Design* (pp. 257–262). https://doi.org/10.1016/B978-0-12-818597-1.50041-2

Five model variants are described below, generated by restricting the module
types available in the original GDP model.
- ``build_model('Gas_100')``: Use only the U100 module size
- ``build_model('Gas_250')``: Use only the U250 module size
- ``build_model('Gas_500')``: Use only the U500 module size
- ``build_model('Gas_small')``: Use the U100 or U250 module sizes
- ``build_model('Gas_large')``: Use the U250 or U500 module sizes

Calling ``build_model()`` without a case builds the unrestricted base model
with all module types.

- ``Gas_100``: -12.34
- ``Gas_250``: -18.37
- ``Gas_500``: -4.69
